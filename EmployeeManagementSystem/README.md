# Employee Management System

Enterprise-grade REST API built for Cognizant Java FSE Deepskilling.

## Tech Stack
* Java 17
* Spring Boot 3.2.0
* Spring Data JPA & Hibernate
* H2 In-Memory Database
* Lombok

## Features Implemented
* Full CRUD REST API for Employees and Departments.
* Bidirectional `@OneToMany` and `@ManyToOne` relationships.
* Entity Auditing (`@CreatedDate`).
* Pagination and Sorting (`Pageable`).
